# NYC Airbnb Analysis & Price Prediction

## Project Overview
This project explores the 2019 NYC Airbnb dataset to identify key factors influencing rental prices. 
As an Industrial Engineering and Management student, I focused on applying data preprocessing and machine learning to derive actionable insights.

## Key Features
* **Data Cleaning:** Handled missing values and removed outliers (e.g., $0 listings) to ensure model accuracy.
* **Exploratory Data Analysis (EDA):** Visualized price distributions and correlations using Seaborn and Matplotlib.
* **Feature Engineering:** Implemented `LabelEncoder` and `MinMaxScaler` for data normalization.
* **Machine Learning:** Built and evaluated Decision Tree and Random Forest models.
* **Feature Importance:** Analyzed which variables (like room type or location) have the most impact on price.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
